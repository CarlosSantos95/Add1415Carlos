1.Configuración de la Máquina Virtual.

Instalaremos un SO Lubuntu,realizaremos una instalación predeterminada.

La máquina virtual que estableceremos como servidor tendrá dos adaptadores de red:
  • Una externa
    • Para conectarnos a internet,la ponemos en adaptador puente con la siguiente IP:172.16.108.11
  • Una interna
    • Para conectarse con los clientes ligeros.La IP de esta interfaz de red debe ser estática y debe estar en la misma red que los clientes (IP 192.168.0.1).

    2.Instalar el servicio LTSP

  • Instalamos el servidor de clientes ligeros
    • apt-get install ltsp-server-standalone
    • ltsp-build-client


  • Instalamos el servidor SSH
    • apt-get install openssh-server



  • Revisamos la configuración de la tarjeta de red interna del servidor. Debemos comprobar que la IP estática sea compatible con la configuración dhcp (/etc/ltsp/dhcpd.conf)
